------------------------------------------------------------
FlowNext – Business, SaaS & App HTML Template
------------------------------------------------------------

Author: Amit Sharma
Version: 1.0
Release Date: November 2025
------------------------------------------------------------

Thank you for purchasing FlowNext!

FlowNext is a modern, responsive multi-purpose HTML template
built with Tailwind CSS and Vanilla JavaScript. It includes
three homepage variations (Business, SaaS, and App) and all
essential inner pages such as About, Services, Pricing, and Contact.

------------------------------------------------------------
Folder Structure
------------------------------------------------------------

flownext-template/
│
├── dist/
│   └── css/
│       └── styles.css          → Compiled Tailwind CSS
│
├── src/
│   ├── index.html              → Home (Business)
│   ├── home-saas.html          → Home (SaaS)
│   ├── home-app.html           → Home (App)
│   ├── about.html
│   ├── services.html
│   ├── pricing.html
│   ├── contact.html
│   ├── js/
│   │   └── main.js             → Global JavaScript
│   └── images/                 → Local images and icons
│
├── documentation/
│   └── documentation.html      → Full usage guide
│
└── README.txt

------------------------------------------------------------
File References
------------------------------------------------------------

Inside each page in /src/ folder:

<link rel="stylesheet" href="../dist/css/styles.css">
<script src="./js/main.js" defer></script>

------------------------------------------------------------
Technologies Used
------------------------------------------------------------

- Tailwind CSS 3.x
- HTML5 / CSS3
- Vanilla JavaScript (no framework)
- AOS (Animate On Scroll) - optional
- Heroicons (MIT License)
- unDraw illustrations (MIT License)
- Unsplash images (Free License)
